Ok so this app is an app that I built to sell to e-commerce businesses, local e-commerce businesses. The app actually is like a template. Whenever I sell to different businesses, I normally change the styling and the name of the store to fit the store or the business owner that I'm trying to sell to.

I want to rebuild it into a Nextjs + TypeScript app, a template app such that whenever I want to sell to a different customer, I just have to tweak a few things. I can add the products that the person is selling and then I can change the name, the design system. I don't want to need to write code to change these things. I want to be able to change the branding throughout the whole app from one place. 

I want to use shadcn/ui for the components and tailwind for the styling.
